# Security Policy

This repository hosts a static personal website (HTML and CSS, no build
step). To report a security issue with the site or its content, please use
[GitHub private vulnerability reporting](https://github.com/rmednitzer/rmednitzer.github.io/security/advisories/new).

A machine-readable security contact is also published at
[`/.well-known/security.txt`](.well-known/security.txt).
